Bottom Element Auto Fill

A Cool Tool

The bottom text-element will fill out the remaining space of the screen without causing scrolling. Originally created for a web-based application where the top had navigation button and a toolbar, while the bottom had a grid of editable, scrollable data (think excel sheet).
